### Fonctions disponibles

Infos :
* Etat du routeur
* Etat du Wifi
* Etat des LED
* Nom du routeur
* Nom des SSiD WiFi
* Version de firmware
* Type de CPU
* Charge système et mémoire

Actions :
* Reboot
* Backup

Scénarii possibles :
* Sauvegarder la configuration à intervales réguliers
